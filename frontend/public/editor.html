Keyboard shortcuts

F1help
ctrl + sSave
ctrl + eEdit mode on/off
ctrl + dDelete line (or pharagraph)
ctrl + qFormat current code block
ctrl + alt + [number]Heading 1-6
All the browser shortcuts work, e.g. ctrl + b for bold (selection or state), and (shift +) tab can be used to (un)indent lists

Commands

imgInsert image
tableInsert table
ulInsert unordered list
olInsert ordered list
h[number]Insert header
Commands work by typing "\command + TAB".
Code block are created by "|language + TAB". The language name should be a class recognized by highlight.js
